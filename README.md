# Mathematica Code for: Vibration response of blue honeysuckle branches based on mathematical model

This repository contains the *Mathematica* notebook (`*.nb`) file used to perform the analytical calculations and generate the figures for the manuscript:

> **Vibration response of blue honeysuckle branches based on mathematical model**  
> *Yuan Wei,Wang Ruiyin,Wang Yecheng,Feng Fang2,Ma Decai*  
> *PLOS ONE* 

## Description
The core of this repository is the file `The instability regions vary with the fruit branch parameters.nb`
and `Effect of branch length variation on system instability region.nb`
and `Explicit dynamics simulation of the fruit branch system.cae`. It implements the Vibration response of blue honeysuckle branches based on mathematical model as described in the manuscript. Key operations include:
- Solving The instability regions vary with the fruit branch parameters (Section 3.1 of the manuscript).
- Calculating the Effect of branch length variation on system instability region (Section 3.1 of the manuscript).
- Explicit dynamics simulation of the fruit branch system (Section 3.12 of the manuscript).

## Requirements
- **Software:** Wolfram *Mathematica* (Version 11.0 or higher recommended). The code has been tested on Version [12.0].
- **Software:**  *Abaqus CAE* (Version 2017 or higher recommended). The code has been tested on Version [12.0].
- **No additional toolboxes or packages are required.**

## Usage
1.  Download or clone this repository.
2.  Open the file `The instability regions vary with the fruit branch parameters.nb`
and `Effect of branch length variation on system instability region.nb` in *Mathematica*.
3.  Execute the notebook cells sequentially (Press `Shift+Enter`).
4.  The main results and plots will be generated automatically. Key parameters (e.g., dimensions, material properties from Table 1) are defined at the beginning of the notebook.
